# FUTURE_CS_03 — Secure File Sharing System

This project implements a secure file sharing system using Python Flask and AES encryption.

## Features

- Secure File Upload
- AES Encryption for stored files
- File Decryption on download
- Simple Web Interface

## Technologies Used

Python
Flask
PyCryptodome
HTML
